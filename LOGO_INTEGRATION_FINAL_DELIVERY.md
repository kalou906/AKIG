## ğŸ“ Support & Maintenance

### Common Issues & Solutions

**Issue**: Logo not showing in PDF
```
Solution:
1. Verify logo file exists at: frontend/public/assets/logos/logo.png
2. Check file permissions (readable by backend process)
3. Review backend logs for errors
4. PDF still generates without logo (graceful fallback)
```

**Issue**: Favicon not updating in browser
```
Solution:
1. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
2. Clear browser cache
3. Verify favicon files deployed to public folder
4. Check manifest.json icon paths
```

**Issue**: Email logo not loading
```
Solution:
1. Verify LOGO_URL environment variable set
2. Test URL is publicly accessible
3. Check email client caching settings
4. Verify HTML email rendering mode enabled
```
